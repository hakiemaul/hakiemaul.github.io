My Projects

REST Man - Restaurant Manager

REST Man is a restaurant management system with the automated reporting system and new menu suggestion as the main features, along with realtime ordering system. This project was built in 7 days as a part of my Hacktiv8 bootcamp's final project with two partners. In this project, I was assigned as the team leader and mostly handled the mobile client side of the application using React Native. Beside developing the Android app, I was also responsible to delegate the tasks, check the team's progress day by day, initiate the daily briefing and managing our git repositories.
